# Feature: Sun Position Arc

## Status: Not Started

## Requirements
- Semicircular arc showing sun position between sunrise and sunset
- Sun icon animates along arc in real-time
- Gradient stroke on traversed arc
- Dashed horizon line
- Sunrise/sunset time labels at endpoints
- Progress: 0.0 (sunrise) to 1.0 (sunset)
- Position math: angle = 180 - (progress * 180), x = center + radius*cos(angle), y = center - radius*sin(angle)
